### `var`, `let`, `const`
- `var`:
  - 变量提升(hoisting)：变量声明(declaration)提升，但初始化(initialization)没有提升！
    ```js
    console.log(a); // undefined
    let a = 5;
    ```
    等价于
    ```js
    var a; // 变量声明提升
    console.log(a);
    a = 5; // 初始化没有提升
    ```
  - 可以对同一个变量多次声明，后者覆盖前者
    ```js
    var a = 1;
    var a = 2;
    console.log(a); // 2
    ```

- `let`/`const`
  - 变量也会“被提升”，但是存在暂时性死区(Temporal Dead Zone)，在执行真正的初始化之前不可以访问！
    ```js
    console.log(b); // ReferenceError
    let b = 10; 
    ```
